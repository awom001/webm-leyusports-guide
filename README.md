# webm-leyusports-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态方式呈现，不依赖特定后端服务，旨在提供结构化的信息展示与查阅功能。

## 项目简介

这是一个以 HTML 页面为核心的项目仓库。所有页面均为独立文件，便于直接通过浏览器打开或部署到任何静态托管服务。仓库内容不针对特定域名或网站，仅作为通用信息归档与展示用途。

## 目录说明

```
webm-leyusports-guide/
├── index.html          # 主页入口
├── pages/              # 存放各独立页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 资源文件夹（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库的首页，提供导航或概览。
- `pages/`：所有独立 HTML 页面存放于此，命名清晰，便于识别。
- `assets/`：页面所需的公共资源，如样式表、脚本文件和图片。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可单独打开或嵌入。
- 页面之间不共享动态状态，所有内容在加载时确定。
- 归档内容以通用性为主，不涉及具体业务或网站数据。
- 页面结构简洁，注重可读性与基础排版。
